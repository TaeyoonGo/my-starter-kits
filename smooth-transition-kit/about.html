About

이 스타터킷의 구조와 사용 방법을 알아보세요.

사용 방법

1. 애니메이션 속성 추가

HTML 요소에 data-animate 또는 data-text-animate 속성을 추가하면 자동으로 애니메이션이 적용됩니다.

2. 유틸리티 함수 사용

JavaScript에서 직접 fadeIn(), charsReveal() 등의 함수를 호출하여 세밀한 제어가 가능합니다.

3. 페이지 전환

Barba.js가 자동으로 페이지 전환을 처리합니다. 일반 링크만 사용하면 됩니다.

애니메이션 옵션

스크롤 애니메이션 (data-animate)

  • fade - 페이드 인
  • slide-left - 왼쪽에서 슬라이드
  • slide-right - 오른쪽에서 슬라이드
  • scale - 스케일 업
  • stagger - 자식 요소 순차 등장

텍스트 애니메이션 (data-text-animate)

  • chars - 글자별 등장
  • words - 단어별 등장
  • lines - 줄별 등장
  • typewriter - 타이핑 효과
  • scatter - 흩어짐 효과
